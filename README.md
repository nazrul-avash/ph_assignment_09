# 🏥 DocAppoint - Doctor Appointment Manager

A modern, full-stack Doctor Appointment Booking System that allows users to browse doctors, book appointments, and manage their healthcare schedule seamlessly.

🔗 **Live Site:** [https://docappoint-five.vercel.app/](https://docappoint-five.vercel.app/)

---

## ✨ Features

- 🔐 **Secure Authentication** — JWT-based auth with Better Auth, including Google/GitHub social login
- 👨‍⚕️ **Browse Doctors** — View top-rated doctors and all available appointments with detailed profiles
- 📅 **Book Appointments** — Easy appointment booking with date/time selection and instant confirmation
- 🛠️ **Manage Bookings** — Update or cancel appointments from your personal dashboard
- ⭐ **Doctor Reviews** — Leave feedback after your appointment (Optional)
- 🌓 **Theme Toggle** — Switch between light and dark mode (Optional)
- 🔍 **Search & Sort** — Find doctors by name and sort appointments (Optional)
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop devices
- 🎨 **Modern UI** — Clean, professional design with Swiper.js sliders and toast notifications

---

## 🛠️ Tech Stack

**Frontend:**
- React 19 + Vite
- React Router DOM
- Tailwind CSS
- Swiper.js
- React Hot Toast

**Backend:**
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- Better Auth

**Deployment:**
- Client: Vercel
- Server: Render
- Database: MongoDB Atlas
